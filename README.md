# air_hockey

## 遊戲介紹
這是一款兩人對戰的遊戲，玩家需要操控自己的搖桿，把冰球推進對方球門就可以得分，最先獲得5分的人就可以取得勝利，在遊戲中可以感受到冰球比賽的激烈氣氛。遊戲畫面簡潔、清晰、逼真，讓玩家可以更沉浸於遊戲之中，享受遊戲帶來的樂趣。
